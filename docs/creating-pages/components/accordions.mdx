A dropdown component to show or hide content.

<Accordion title="I'm an Accordion">
You can add any content in here. Check out <a href="#accordion-group">Accordion Group</a> if you want to group multiple Accordions into a single display.
</Accordion>

This is how you build an Accordion:
```jsx
<Accordion title="Code for an Accordion">
Description for an Accordion.
</Accordion>
```
### Attributes

<strong>title</strong> <small className="small_tag"><code>string</code></small> <small className="required_tag"><code>required</code></small>
<br/>Title displayed in the Accordion preview.

<strong>description</strong> <small className="small_tag"><code>string</code></small> <small className="required_tag"><code>required</code></small>
<br/>Detail text displayed below the title in the Accordion preview.

## Accordion Group
To group multiple <code>&lt;Accordion&gt;</code> elements into a single display, simply enclose them within the <code>&lt;AccordionGroup&gt;</code> tag.

<AccordionGroup>

  <Accordion title="Accordion with Callout">
  Feel free to add other components inside an Accordion, like Callouts. 
    <Callout type="tip" title="Tip">
    Accordions are great show or hide content.
    </Callout>  
  </Accordion>

  <Accordion title="Accordion with Code Block">
  You can also add code blocks here.
  
    ```js
    const axios = require('axios');

    let config = {
     method: 'get',
     maxBodyLength: Infinity,
     url: 'https://api.example.com/v1/items',
     headers: { 
        'Accept': 'application/json'
     }
    };
```
  </Accordion>

</AccordionGroup>
